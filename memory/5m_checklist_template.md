- [ ] **现金转换周期**
  - 应收账款天数:

- [ ] **自由现金流质量**
  - 净利润现金含量(经营现金流/净利润):

  - 硬核ROE = 现金利润/净资产:

- [ ] **颠覆风险评估**
  - [ ] 技术颠覆风险: 高/中/低
  - [ ] 监管风险: 高/中/低
  - [ ] 替代品威胁: 高/中/低

### M4 评分: ___/10

**评分标准**:
- 10分：轻资产，现金周期短，FCF优秀，无颠覆风险
- 7-9分：商业模式清晰，现金流健康，风险可控
- 4-6分：商业模式一般，现金流波动，存在一定风险
- 1-3分：重资产，现金流差，面临重大颠覆风险 ⚠️ 空头信号
