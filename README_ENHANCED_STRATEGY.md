# 增强版比特币交易策略（集成恐惧贪婪指数和AHR999指数）

此系统实现了增强版的自动化比特币交易策略，不仅包含基础技术指标，还集成了市场情绪指标（恐惧贪婪指数）和AHR999估值指标。

## 新增特性

1. **恐惧贪婪指数集成**：从Alternative.me获取每日市场情绪数据
2. **AHR999估值指标**：基于200日移动平均线的比特币估值指标
3. **综合信号判断**：结合技术指标、市场情绪和估值指标进行决策

## 策略逻辑

### 恐惧贪婪指数解读
- **0-20**: 极度恐惧 → 可能的买入机会
- **20-40**: 恐惧 → 谨慎买入
- **40-60**: 中性 → 观望
- **60-80**: 贪婪 → 谨慎持有
- **80-100**: 极度贪婪 → 考虑卖出

### AHR999指数解读
- **< 0.4**: 显著低估 → 强烈买入信号
- **0.4-0.8**: 低估 → 买入信号
- **0.8-1.2**: 合理估值 → 持有
- **> 1.2**: 高估 → 卖出信号

### 综合决策机制
系统会同时分析以下三个维度：
1. **技术指标**：SMA、RSI、MACD、布林带等
2. **市场情绪**：恐惧贪婪指数
3. **估值水平**：AHR999指数

然后根据三个维度的一致性来确定最终推荐：
- 当至少两个指标指向同一方向时，提高置信度
- 所有指标综合评估后给出BUY/SELL/HOLD建议
