# Friday Portfolio - 模拟投资组合

**目标**: 年化收益 20%+  
**策略**: 大仓稳健，小仓爆发，攻守兼备  
**初始资金**: ¥1,000,000  
**开始日期**: 2026-02-09

> **责任声明**: Friday (AI) 对本模拟投资组合的资金全权负责。所有交易决策基于独立分析，承担相应责任。

---

## 🎯 投资责任框架

### 决策责任
- **独立分析**: 基于晨报、数据、逻辑自主决策
- **完整记录**: 每笔交易记录决策上下文和理由
- **风险控制**: 严格遵守仓位管理和止损规则
- **定期复盘**: 每周回顾组合表现，优化策略

### 风控红线
| 规则 | 限制 | 违反后果 |
|------|------|----------|
| 单只核心标的 | ≤ 20% | 超额部分强制减仓 |
| 单只卫星标的 | ≤ 8% | 拒绝执行 |
| 进攻型合计 | ≤ 60% | 触发防守模式 |
| 单日回撤 | > 5% | 暂停买入，评估风险 |
| 止损线 | -15% | 强制止损 |

### 复盘机制
- **每日**: 收盘后检查持仓表现
- **每周**: 周末复盘，更新策略
- **每月**: 月度总结，调整仓位结构
- **每季**: 深度复盘，优化选股框架

### 责任追溯
所有决策记录永久保存：
- 交易记录 (`data/trades.json`)
- 决策逻辑 (交易中的 `context` 和 `logic`)
- 复盘笔记 (`memory/` 目录)

## 💰 交易执行规则

> ⚠️ **重要**: 本模拟盘仅由 **Friday (AI)** 进行操作，用户仅可查看，不可下达交易指令。

### 交易执行主体
- **唯一操作人**: Friday (AI)
- **用户权限**: 查看、监督、建议（但不可指令交易）
- **决策独立性**: AI 根据市场分析自主决策

### 交易触发方式
1. **晨报分析后** - 发现机会时自主执行
2. **盘中异动** - 满足预设条件时自动触发
3. **定期复盘** - 每周调整优化

### 通知机制
每笔交易执行后：
- ✅ 自动同步到网站
- ✅ 发送 Telegram 交易通知
- ✅ 记录完整决策逻辑
